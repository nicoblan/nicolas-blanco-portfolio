# Portfolio - React + Vite + Tailwind

## Setup
1. `npm install`
2. `npm run dev` (abre http://localhost:5173)

## Notas
- Reemplaza `ACTION_FORM` en `/src/components/Contact.jsx` por tu endpoint de Formspree (p. ej. `https://formspree.io/f/XXXXX`) o usa otro servicio.
- Personaliza `src/components/*` con tus proyectos, links y fotos.
- Para deploy en Vercel: conecta tu repo y se deploya automáticamente.

Si querés, puedo:
- Añadir ESLint / Prettier.
- Integrar GitHub Actions para deploy automático.
- Reemplazar imágenes por las tuyas y agregar tus links al README.
